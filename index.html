Enter Custom Pattern
Type your solfege pattern using the shorthand notation below:
Syntax:
d=do, r=re, m=mi, f=fa, s=so, l=la, t=ti
Add , for low octave (e.g., s,=so,)
Add ' for high octave (e.g., d'=do')

Example: ds,dd' = do, so,, do, do'
